# Rust Colordle

A color guessing game made using Rust.

## About

A simple color guessing game with over 30 thousand different named colors. 

Color names from the [color pizza api](https://api.color.pizza/v1/docs/)

GUI made with [egui](https://docs.rs/egui/latest/egui/index.html)

Created using `rustc 1.90.0`

## Planned features

- New mode like the original wordle where you guess the hex value of a color.
- Different color palettes and difficulty modes
- Different types of hints: number of words, example of color, hex values, etc.
- Wins tracker and persistent data
